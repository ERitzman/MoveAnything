## Latest version from CurseForge original author, prior to 4.0 implementations
This is a clone of the Original CurseForge version of MoveAnything.
-------------------------------------------------------------
Allows you to move ui elements to your prefered location
example: move your portrait to the bottom so that it is more visible to you.

I am placing this here because the ui elements in Ascension Bronzebeard seem to always stack on top of one another.
